# 股票池维护说明

股票池在 [README.md](README.md) 维护，pipeline.py 会自动读取。每行一只，按市场分组。

## 格式

`代码 名称`（空格分隔，股本自动从腾讯证券接口获取，无需手填）

| 市场 | 分组标题 | 代码示例 | 说明 |
|------|----------|----------|------|
| A股  | `## A股` | `600519` | 股本自动查（腾讯/东方财富多源兜底） |
| 港股 | `## 港股` | `00700`  | 股本自动查（腾讯证券） |
| 美股 | `## 美股` | `AAPL`   | 股本自动查（腾讯证券） |

## 境外股票用什么名字

显示名一律用**中文**（如 `苹果`、`腾讯控股`、`英伟达`），代码用**市场本位代码**（`AAPL`、`00700`、`NVDA`），与东方财富一致。表格里都是中文名，统一好认。

## 币种说明

市值 = 股价 × 股本，自然在股价币种 (A股 CNY / 港股 HKD / 美股 USD)。
利润保留原始报告币种 (A股/港股红筹 CNY、美股本土 USD、中概股 CNY)。
PE 内部统一币种计算，不额外折算。

## 添加新股票

在对应分组下加一行即可：

```
- 代码 名称
```

推送到 main 后 GitHub Actions 会自动更新数据。

> 股票池格式极简：只需 `代码 名称`，股本/价格/市值全部自动获取。
> 腾讯证券接口 (`qt.gtimg.cn`) 三市场统一，不限流，作为主行情源。
