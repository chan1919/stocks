# 股票池维护说明

股票池在 [README.md](README.md) 维护，pipeline.py 会自动读取。每行一只，按市场分组。

## 格式

`代码 中文名 [股本]`（空格分隔，股本可选，单位:亿股）

| 市场 | 分组标题 | 代码示例 | 是否要写股本 | 说明 |
|------|----------|----------|-------------|------|
| A股  | `## A股` | `600519` | 否（自动查） | 股本自动从行情接口获取最新值，送股/拆股/增发自动跟进；查不到时用本地缓存兜底 |
| 港股 | `## 港股` | `00700`  | **是** | 港股无公开股本查询接口，必须手填股本(亿股) |
| 美股 | `## 美股` | `AAPL`   | **是** | 美股同理，必须手填股本(亿股) |

## 境外股票用什么名字

显示名一律用**中文**（如 `苹果`、`腾讯控股`、`英伟达`），代码用**市场本位代码**（`AAPL`、`00700`、`NVDA`），与东方财富一致。表格里都是中文名，统一好认。

## 币种说明

市值 = 股价 × 股本，自然在股价币种 (A股 CNY / 港股 HKD / 美股 USD)。
利润保留原始报告币种 (A股/港股红筹 CNY、美股本土 USD、中概股 CNY)。
PE 内部统一币种计算，不额外折算。

显示名一律用**中文**（如 `苹果`、`腾讯控股`、`英伟达`），代码用**市场本位代码**（`AAPL`、`00700`、`NVDA`），与东方财富一致。表格里都是中文名，统一好认。

## 添加新股票

在对应分组下加一行即可：

- A 股：`- 代码 中文名`
- 港股 / 美股：`- 代码 中文名 股本`

推送到 main 后 GitHub Actions 会自动更新数据。

> 股本缓存文件 `data/shares_cache.json` 为 A 股股本兜底，首次运行会自动生成，请一并提交。发生送股/拆股/增发时，港股美股需手动改 README 对应行；A 股会自动更新。
